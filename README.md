# python-django-sample
## 環境構築（Windows）
* Djangoモジュールのインストール
```
pip install django
```
* 新規プロジェクトの作成
```
py -m django startproject pyFirst
```

## プロジェクト
* pyFirst
    * viewを関数ベースで定義
* pySecond
    * viewを汎用View Classで定義
